# 日总结功能

## 设计原则
- 数据长久保留，支持历史查询
- 核心指标突出：订单数、各菜品销量
- 界面简洁，一目了然

## 数据维度

### 核心统计数据

| 统计项 | 说明 |
|--------|------|
| 订单总数 | 当日下单次数（每张票算一单） |
| 各菜品销量 | 每种菜品的销售次数 |
| 时间段分布 | 按小时段展示订单分布（可选展示） |

### 不统计项

- 营业额（因价格为可选项，不计算）
- 客单价（同上）

## 数据存储

- 使用 SQLite 持久化存储
- 数据长久保留，不自动清理
- 按日期分区查询，保证查询效率

## 日总结数据结构

```dart
class DailySummary {
  final DateTime date;                    // 日期
  final int totalOrders;                  // 订单总数
  final List<DishSummary> dishSummaries;  // 各菜品销量
  final Map<int, int> hourlyDistribution; // 时间段分布（小时:订单数）
}

class DishSummary {
  final int dishId;       // 菜品ID
  final String dishName;  // 菜品名称
  final int count;        // 销售数量
}
```

### 日期选择器

- 左右箭头切换上一天/下一天
- 点击日期弹出日历选择器
- 默认展示当天数据
- 未来日期不可选择

## 功能清单

### 1. 日期切换
- [x] 左右箭头切换日期
- [x] 点击日期弹出日历选择
- [ ] 快速返回今天按钮

### 2. 数据展示
- [x] 显示当日订单总数（大字突出）
- [x] 按销量排序展示各菜品销量
- [x] 可折叠的时间段分布图

### 3. 数据导出（可选扩展）
- [ ] 导出某日数据为文本/CSV
- [ ] 分享统计数据

## 界面交互

| 操作 | 效果 |
|------|------|
| 点击左箭头 | 切换到前一天 |
| 点击右箭头 | 切换到后一天（不超过今天） |
| 点击日期 | 弹出日历选择器 |
| 点击"时段分布" | 展开/折叠时间段图表 |
| 长按某日数据 | 弹出菜单（可选：导出、分享） |
