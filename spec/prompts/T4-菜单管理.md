## 详细步骤

### 1. MenuService (`lib/services/menu_service.dart`)
```dart
class MenuService {
  final DishDao _dishDao;

  MenuService(this._dishDao);

  Future<List<Dish>> getAllDishes();
  Future<Dish> addDish(String name, double? price, String? imagePath);
  Future<void> updateDish(Dish dish);
  Future<void> deleteDish(int id);
  Future<void> reorderDishes(List<Dish> dishes);  // 更新排序
}
```

### 2. MenuProvider (`lib/providers/menu_provider.dart`)
使用 Provider 管理状态：
```dart
class MenuProvider extends ChangeNotifier {
  final MenuService _menuService;
  List<Dish> _dishes = [];
  bool _isLoading = false;

  // Getters...

  Future<void> loadDishes();
  Future<void> addDish(...);
  Future<void> updateDish(...);
  Future<void> deleteDish(int id);
  Future<void> reorderDishes(int oldIndex, int newIndex);
}
```

### 3. MenuManagementScreen (`lib/screens/menu_management_screen.dart`)
界面要求：
- 顶部标题栏 + 添加按钮
- 网格展示菜品（2-3列，根据屏幕宽度自适应）
- 每个菜品卡片显示：名称、价格（如有）、图片（如有）
- 长按弹出菜单：编辑、删除
- 支持拖拽排序（ReorderableGridView 或类似）

### 4. DishEditScreen (`lib/screens/dish_edit_screen.dart`)
表单字段：
- 名称（必填，TextFormField with validator）
- 价格（可选，数字输入）
- 图片（可选，点击选择图片，使用 image_picker）
- 保存按钮

两种模式：
- 新增模式：标题"添加菜品"
- 编辑模式：标题"编辑菜品"，填充已有数据

### 5. DishGridItem (`lib/widgets/dish_grid_item.dart`)
可复用的菜品网格项组件：
```dart
class DishGridItem extends StatelessWidget {
  final Dish dish;
  final VoidCallback? onTap;
  final VoidCallback? onLongPress;

  // 显示图片（如有）或占位图标
  // 名称和价格

```

## 技术要求
- 使用 ChangeNotifierProvider
- 表单验证：名称为空时显示错误提示
- 删除前显示确认对话框
- 图片使用 File 存储本地路径
- 排序后更新 sort_order 字段
