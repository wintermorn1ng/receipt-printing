# T10: 代码整理与文档

## 任务目标
代码审查、重构、补充注释，确保代码质量。

## 代码审查清单

### 代码规范
- [ ] 命名规范（类名大驼峰，变量小驼峰，常量全大写）
- [ ] 文件组织（每个类单独文件）
- [ ] 导入排序（dart:flutter, package, relative）
- [ ] 无用导入清理
- [ ] 代码格式化（dart format）

### 错误处理
- [ ] 所有异步操作有 try-catch
- [ ] 数据库操作有错误处理
- [ ] 蓝牙操作有错误处理
- [ ] 用户输入有验证

### 资源管理
- [ ] 数据库连接正确关闭
- [ ] 蓝牙连接正确断开
- [ ] Stream 正确取消订阅
- [ ] 图片资源正确释放

### 性能优化
- [ ] 列表使用 ListView.builder
- [ ] 图片使用缓存
- [ ] 避免不必要的 rebuild
- [ ] 数据库查询有索引

## 注释规范

### 文件头注释
```dart
/// 功能描述
///
/// 作者：[开发agent]
/// 日期：2024-03-01
```

### 类注释
```dart
/// 菜单服务类
///
/// 负责菜品的增删改查和排序管理
class MenuService {
```

### 方法注释
```dart
  /// 下单并打印小票
  ///
  /// [dishId] 菜品ID
  /// [dishName] 菜品名称
  ///
  /// 返回创建的订单对象
  ///
  /// 可能抛出 [PrintException] 当打印失败时
  Future<Order> placeOrder(int dishId, String dishName);
```

## 重构检查项

1. **提取重复代码**
   - 检查是否有复制粘贴的代码块
   - 提取为公共方法或组件

2. **简化复杂方法**
   - 方法行数不超过 50 行
   - 嵌套层级不超过 3 层

3. **常量提取**
   - 魔法数字提取为具名常量
   - 重复字符串提取为常量

4. **状态管理优化**
   - 检查是否有不必要的 Provider
   - 确保状态更新最小化

## 最终检查

运行命令检查：
```bash
# 代码格式
flutter format --set-exit-if-changed lib/

# 静态分析
flutter analyze

# 构建测试
flutter build apk --release
```
