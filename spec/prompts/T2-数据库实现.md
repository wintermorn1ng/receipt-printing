## 数据库表结构

### dishes 表
```sql
CREATE TABLE dishes (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  price REAL,
  image_path TEXT,
  sort_order INTEGER DEFAULT 0,
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL
);
```

### orders 表
```sql
CREATE TABLE orders (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  ticket_number INTEGER NOT NULL,
  dish_id INTEGER NOT NULL,
  dish_name TEXT NOT NULL,
  created_at INTEGER NOT NULL,
  date TEXT NOT NULL  -- 格式: YYYY-MM-DD，用于快速统计
);
CREATE INDEX idx_orders_date ON orders(date);
```

### settings 表
```sql
CREATE TABLE settings (
  key TEXT PRIMARY KEY,
  value TEXT
);
```

## 详细步骤

1. **DatabaseHelper** (`lib/database/database_helper.dart`)
   - 单例模式
   - 数据库初始化（onCreate）
   - 数据库升级（onUpgrade）
   - 获取数据库实例方法

2. **DishDao** (`lib/database/dish_dao.dart`)
   - `Future<List<Dish>> getAll()` - 获取所有菜品
   - `Future<Dish?> getById(int id)` - 根据ID获取
   - `Future<int> insert(Dish dish)` - 插入
   - `Future<int> update(Dish dish)` - 更新
   - `Future<int> delete(int id)` - 删除
   - `Future<void> updateSortOrder(List<Dish> dishes)` - 批量更新排序

3. **OrderDao** (`lib/database/order_dao.dart`)
   - `Future<int> insert(Order order)` - 插入订单
   - `Future<List<Order>> getByDate(String date)` - 获取某日订单
   - `Future<int> getOrderCountByDate(String date)` - 获取某日订单数
   - `Future<Map<String, int>> getDishCountByDate(String date)` - 获取某日各菜品销量
   - `Future<Map<int, int>> getHourlyDistribution(String date)` - 获取时段分布

## 技术要求
- 使用 sqflite 包
- 日期存储为 INTEGER (millisecondsSinceEpoch)
