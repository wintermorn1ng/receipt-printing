### 3. 蓝牙设备列表
- 显示已配对设备和搜索到的新设备
- 已连接设备显示绿色勾选
- 点击未连接设备进行配对/连接
- 连接中显示 loading

### 4. 小票设置
- 打印店名：Switch + 店名输入框
- 打印日期时间：Switch
- 打印两联：Switch
- 实时保存到 PrinterConfig

### 5. 自动重连逻辑
在应用启动时（main.dart）：
```dart
final printService = PrintService();
final config = await printService.getPrinterConfig();
if (config.deviceAddress != null) {
  // 尝试自动连接，失败不阻塞
  printService.connect(config.deviceAddress!).catchError((_) {
    // 静默失败，首页显示未连接状态
  });
}
```

### 6. 主页显示连接状态
在 `HomeScreen` 顶部（取餐号旁边）显示蓝牙图标：
- 已连接：蓝色蓝牙图标
- 未连接：灰色蓝牙图标 + 点击跳转设置

## 技术要求
- 需要蓝牙权限（AndroidManifest.xml）：
  ```xml
  <uses-permission android:name="android.permission.BLUETOOTH" />
  <uses-permission android:name="android.permission.BLUETOOTH_ADMIN" />
  <uses-permission android:name="android.permission.ACCESS_FINE_LOCATION" />
  <uses-permission android:name="android.permission.BLUETOOTH_CONNECT" />
  <uses-permission android:name="android.permission.BLUETOOTH_SCAN" />
  ```
- 运行时权限申请（Android 12+ 需要 BLUETOOTH_CONNECT）
- 搜索超时：10秒
- 连接超时：5秒
