### 3. 错误处理
定义 PrintException：
```dart
enum PrintError {
  bluetoothNotEnabled,    // 蓝牙未开启
  printerNotConnected,    // 打印机未连接
  printerDisconnected,    // 打印中断开
  printFailed,            // 打印指令失败
  paperOut,               // 缺纸
}

class PrintException implements Exception {
  final PrintError error;
  final String message;

  PrintException(this.error, this.message);
}
```

### 4. 集成到 OrderService
修改 `lib/services/order_service.dart`：
```dart
Future<Order> placeOrder(int dishId, String dishName) async {
  // 1. 获取取餐号
  final ticketNumber = await _ticketService.getCurrentTicketNumber();

  // 2. 创建订单
  final order = Order(...);
  await _orderDao.insert(order);

  // 3. 打印小票（可能失败，但不影响订单创建）
  try {
    final config = await _printService.getPrinterConfig();
    await _printService.printTwoCopies(order, config);
  } catch (e) {
    // 打印失败抛出异常，让UI层处理
    throw PrintException(...);
  }

  // 4. 递增取餐号
  await _ticketService.incrementTicketNumber();

  return order;
}
```

### 5. UI 层错误处理
在 `HomeScreen` 中处理打印错误：
```dart
try {
  await orderProvider.placeOrder(dish);
  showToast('打印成功');
} on PrintException catch (e) {
  showErrorDialog(e.message, e.error);
}
```

错误弹窗内容：
- 蓝牙未开启 → "请开启蓝牙"
- 打印机未连接 → "请前往打印设置连接打印机"
- 其他错误 → 显示具体原因

## 技术要求
- 使用 flutter_bluetooth_serial 包
- ESC/POS 指令参考：初始化、字体大小、对齐、切纸
- 字体大小：取餐号用双倍宽高（0x1D 0x21 0x11）
- 连接失败要有超时处理（5秒）
- 打印异常不影响订单数据保存
