## 架构设计

采用渲染器模式，抽象打印接口，支持多种渲染目标：

```
PrintRenderer (抽象基类)
├── ESCPOSRenderer (蓝牙打印机渲染器)
└── PreviewRenderer (页面预览渲染器)
```

### 1. 抽象打印渲染器 (`lib/utils/print_renderer.dart`)

```dart
/// 打印数据模型
class PrintData {
  final int ticketNumber;
  final String dishName;
  final String? shopName;
  final DateTime? dateTime;

  PrintData({
    required this.ticketNumber,
    required this.dishName,
    this.shopName,
    this.dateTime,
  });
}

/// 抽象打印渲染器
abstract class PrintRenderer {
  /// 渲染单张小票
  Future<void> render(PrintData data);

  /// 渲染两联小票
  Future<void> renderTwoCopies(PrintData data);

  /// 释放资源
  Future<void> dispose();
}
```

### 3. 页面预览渲染器 (`lib/utils/preview_renderer.dart`)

```dart
class PreviewRenderer implements PrintRenderer {
  final StreamController<List<PreviewLine>> _linesController =
      StreamController<List<PreviewLine>>.broadcast();

  /// 预览数据流，供 UI 订阅
  Stream<List<PreviewLine>> get linesStream => _linesController.stream;

  @override
  Future<void> render(PrintData data) async {
    // 生成预览线条数据
    final lines = _generatePreviewLines(data);
    _linesController.add(lines);
  }

  @override
  Future<void> renderTwoCopies(PrintData data) async {
    // 预览时只显示一联
    await render(data);
  }

  List<PreviewLine> _generatePreviewLines(PrintData data) {
    // 生成预览用的线条列表
    return [
      if (data.shopName != null)
        PreviewLine(data.shopName!, alignment: TextAlign.center),
      PreviewLine('-' * 20, alignment: TextAlign.center),
      PreviewLine('#${data.ticketNumber}',
          alignment: TextAlign.center,
          isLarge: true,
          isBold: true),
      PreviewLine('-' * 20, alignment: TextAlign.center),
      PreviewLine(data.dishName, alignment: TextAlign.center),
      PreviewLine(''),
      if (data.dateTime != null)
        PreviewLine(_formatDateTime(data.dateTime!), alignment: TextAlign.center),
      PreviewLine(''),
    ];
  }

  @override
  Future<void> dispose() async {
    await _linesController.close();
  }
}

/// 预览线条模型
class PreviewLine {
  final String text;
  final TextAlign alignment;
  final bool isLarge;   // 大字（取餐号）
  final bool isBold;    // 粗体

  PreviewLine(this.text, {
    this.alignment = TextAlign.center,
    this.isLarge = false,
    this.isBold = false,
  });
}
```

### 6. 错误处理
定义 PrintException：
```dart
enum PrintError {
  bluetoothNotEnabled,    // 蓝牙未开启
  printerNotConnected,    // 打印机未连接
  printerDisconnected,    // 打印中断开
  printFailed,            // 打印指令失败
  paperOut,               // 缺纸
}

class PrintException implements Exception {
  final PrintError error;
  final String message;

  PrintException(this.error, this.message);
}
```

Future<Order> placeOrder(int dishId, String dishName) async {
  // 1. 获取取餐号
  final ticketNumber = await _ticketService.getCurrentTicketNumber();

  // 2. 创建订单
  final order = Order(...);
  await _orderDao.insert(order);

  // 3. 打印小票（可能失败，但不影响订单创建）
  try {
    final config = await _printService.getPrinterConfig();
    await _printService.printTwoCopies(order, config);
  } catch (e) {
    // 打印失败抛出异常，让UI层处理
    throw PrintException(...);
  }

  // 4. 递增取餐号
  await _ticketService.incrementTicketNumber();

  return order;
}

### 8. 预览页面 (`lib/screens/print_preview_screen.dart`)

```dart
class PrintPreviewScreen extends StatefulWidget {
  final Order order;

  const PrintPreviewScreen({super.key, required this.order});

  @override
  State<PrintPreviewScreen> createState() => _PrintPreviewScreenState();
}

class _PrintPreviewScreenState extends State<PrintPreviewScreen> {
  late final PreviewRenderer _previewRenderer;
  late final PrintService _printService;

  @override
  void initState() {
    super.initState();
    _previewRenderer = PreviewRenderer();
    _printService = PrintService(_previewRenderer);

    // 初始化预览数据
    _loadPreview();
  }

  Future<void> _loadPreview() async {
    final printData = PrintData(
      ticketNumber: widget.order.ticketNumber,
      dishName: widget.order.dishName,
      shopName: await _getShopName(),
      dateTime: widget.order.createdAt,
    );
    await _previewRenderer.render(printData);
  }

  @override
  void dispose() {
    _previewRenderer.dispose();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(title: const Text('打印预览')),
      body: StreamBuilder<List<PreviewLine>>(
        stream: _previewRenderer.linesStream,
        builder: (context, snapshot) {
          if (!snapshot.hasData) {
            return const Center(child: CircularProgressIndicator());
          }

          return Container(
            margin: const EdgeInsets.all(16),
            padding: const EdgeInsets.all(16),
            decoration: BoxDecoration(
              color: Colors.white,
              border: Border.all(color: Colors.grey),
              borderRadius: BorderRadius.circular(8),
            ),
            child: Column(
              mainAxisSize: MainAxisSize.min,
              children: snapshot.data!
                  .map((line) => _buildPreviewLine(line))
                  .toList(),
            ),
          );
        },
      ),
      bottomNavigationBar: BottomActionBar(
        onPrint: _handlePrint,
      ),
    );
  }

  Widget _buildPreviewLine(PreviewLine line) {
    return Padding(
      padding: const EdgeInsets.symmetric(vertical: 2),
      child: Text(
        line.text,
        textAlign: line.alignment,
        style: TextStyle(
          fontSize: line.isLarge ? 24 : 14,
          fontWeight: line.isBold ? FontWeight.bold : FontWeight.normal,
        ),
      ),
    );
  }

  Future<void> _handlePrint() async {
    // 切换到真正的打印机渲染器
    final escposRenderer = ESCPOSRenderer();
    _printService.setRenderer(escposRenderer);

    try {
      await _printService.printTwoCopies(widget.order, await _getPrinterConfig());
      if (mounted) showToast('打印成功');
    } catch (e) {
      if (mounted) showErrorDialog(e.toString());
    } finally {
      escposRenderer.dispose();
    }
  }
}
```

## 技术要求
- 使用 flutter_bluetooth_serial 包
- ESC/POS 指令参考：初始化、字体大小、对齐、切纸
- 字体大小：取餐号用双倍宽高（0x1D 0x21 0x11）
- 连接失败要有超时处理（5秒）
- 打印异常不影响订单数据保存
- 渲染器模式便于单元测试（可以轻松 mock PrintRenderer）
