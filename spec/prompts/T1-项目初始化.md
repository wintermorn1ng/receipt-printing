# T1: 项目初始化与依赖配置

## 任务目标
创建 Flutter 项目，配置必要依赖，建立基础目录结构。

## 前置条件
- Flutter SDK 已安装（flutter doctor 通过）
- 当前目录: /Users/victor/Documents/codes/flutter/receipt-printing

## 详细步骤

1. **创建 Flutter 项目**
   ```bash
   flutter create --project-name receipt_printing --org com.example .
   ```

2. **配置 pubspec.yaml**
   添加以下依赖：
   - sqflite: ^2.3.0
   - path_provider: ^2.1.1
   - flutter_bluetooth_serial: ^0.4.0
   - provider: ^6.1.1
   - intl: ^0.18.1

3. **创建目录结构**
   ```
   lib/
   ├── main.dart
   ├── app.dart
   ├── models/
   ├── database/
   ├── services/
   ├── providers/
   ├── screens/
   ├── widgets/
   │   └── common/
   └── utils/
   ```

4. **编写 main.dart**
   - 应用入口
   - Provider 顶层注入

5. **编写 app.dart**
   - MaterialApp 配置
   - 主题设置（简洁风格）
   - 首页路由

## 输出文件
- `pubspec.yaml`
- `lib/main.dart`
- `lib/app.dart`
- 空目录结构
