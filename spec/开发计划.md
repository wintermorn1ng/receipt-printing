## 详细任务描述

### T1: 项目初始化与依赖配置

**目标**：创建 Flutter 项目，配置必要依赖

**输入**：
- `设计spec.md`
- `菜单功能 spec.md`
- `点单功能 spec.md`
- `小票打印功能 spec.md`

**输出**：
- `pubspec.yaml`（配置依赖）
- `lib/main.dart`
- `lib/app.dart`
- 项目基础目录结构

**技术要点**：
- sqflite: ^2.3.0 (数据库)
- path_provider: ^2.1.1 (路径)
- flutter_bluetooth_serial: ^0.4.0 (蓝牙)
- provider: ^6.1.1 (状态管理)
- intl: ^0.18.1 (日期格式化)

---

### T2: 数据库设计与实现

**目标**：实现 SQLite 数据库层，包含所有表结构和基础 CRUD

**输入**：
- `设计spec.md`
- 各模块 spec 中的数据结构定义

**输出**：
- `lib/database/database_helper.dart`
- `lib/database/dish_dao.dart`
- `lib/database/order_dao.dart`

**数据库表结构**：

```sql
-- dishes 表
CREATE TABLE dishes (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  name TEXT NOT NULL,
  price REAL,
  image_path TEXT,
  sort_order INTEGER DEFAULT 0,
  created_at INTEGER NOT NULL,
  updated_at INTEGER NOT NULL
);

-- orders 表
CREATE TABLE orders (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  ticket_number INTEGER NOT NULL,
  dish_id INTEGER NOT NULL,
  dish_name TEXT NOT NULL,
  created_at INTEGER NOT NULL,
  date TEXT NOT NULL  -- 格式: YYYY-MM-DD，用于快速统计
);

-- settings 表（存储取餐号、打印机配置等）
CREATE TABLE settings (
  key TEXT PRIMARY KEY,
  value TEXT
);
```
