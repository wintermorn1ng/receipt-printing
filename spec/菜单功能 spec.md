# 菜单功能

## 设计原则
- 极致简洁，拉平结构，无分类层级
- 核心信息突出，可选信息轻量化

## 菜品数据结构

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | int | 是 | 唯一标识 |
| name | String | 是 | 菜品名称，核心信息 |
| price | double | 否 | 价格，可选项 |
| image | String | 否 | 图片路径，可选项 |
| sort_order | int | 是 | 排序顺序 |
| created_at | DateTime | 是 | 创建时间 |
| updated_at | DateTime | 是 | 更新时间 |

## 功能清单

### 1. 菜品列表展示
- 以网格形式展示所有菜品
- 无分类筛选，所有菜品平铺展示
- 支持自定义排序（拖拽调整顺序）

### 2. 菜品管理
- 新增菜品：填写名称、价格（可选）、上传图片（可选）
- 编辑菜品：修改已有菜品信息
- 删除菜品：删除前需确认
- 排序调整：长按拖拽调整展示顺序

### 3. 数据持久化
- 使用 SQLite 本地存储
- 数据库文件位于应用私有目录
- 启动时自动加载菜单数据到内存
